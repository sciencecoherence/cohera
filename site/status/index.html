Interpretation guide

Green workflow checks with stale content means ingestion is idle; red workflow checks means build/deploy failure. Use Pipeline Control to trigger a manual cycle and inspect logs.

Publish mode: targeted publish only. The pipeline rebuilds and deploys only affected publication artifacts/indexes, not the full site, unless core layout files changed.
