## Core Architecture
The site is divided into a main research hub and a separated trading microsite.

### 1. Research Hub (Science Coherence)
**Root**: `/site/`
**Threads**:
- **Cosmos** (`/site/cosmos/`): Physics, model formalization, time crystals.
- **Regenesis** (`/site/regenesis/`): Biology, sleep protocols, nutrition.
- **Ethos** (`/site/ethos/`): Epistemology, research literacy.

**Shared Resources**:
- **Publications**: `/site/publications/` (Findings reports, changelogs, audits).
- **Glossary**: `/site/glossary/`.
- **Contact**: `/site/contact/`.
- **Assets**: `/site/assets/style.css` (Main "Premium" design).

### 2. Trading Microsite (Cohera Trading)
**Root**: `/site/trading/`
**Design**: Uses independent `/site/trading/assets/style.css`.
**Pages**:
- Playbook: `/site/trading/btc-playbook-v1.html`
- Journal: `/site/trading/journal/index.html`
- Signals: `/site/trading/signals/index.html`

## URL & Path Rules
- **Deployment Base**: The site is deployed at `/cohera/`.
- **Internal Links**: Must account for the base path.
    - Example: `<a href="/cohera/cosmos/index.html">` (Correct for deployed site).
    - Avoid relative `../../` where possible to prevent breakage if files move.

## Content Conventions
- **Digests**: stored in `/site/<thread>/digests/<slug>.html`.
- **Feeds**: `/site/<thread>/digests/index.json`.
- **Autodrafts**: Generated by scripts into the relevant digest folder with `auto-` prefix.

## Quality Constraints
- **Design**: Main site uses the "Premium" dark theme (`--bg: #080b12`).
- **Responsiveness**: All pages must be mobile-friendly.
- **Meta**: Every page needs `<title>` and `<meta name="description">`.
